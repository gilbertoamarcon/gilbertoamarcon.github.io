COLIN2 Wrapper


Wrapper for COLIN2 Planning.

Usage: ./colin2.sh problem-dir

Generating problem file
python gen-problem.py $1

Planning
~/dev/colin2/plan $1/domain.pddl $1/problem.pddl $1/solution $1/timing

VAL PDF
${TOOLS}val-report.sh $1

Generating Graphics
python plot-plan.py $1

Viewing Graphics
evince $1/graphics.eps &
