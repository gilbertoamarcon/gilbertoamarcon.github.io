Basic Desktop Setup


This script sets up all the apps and basic configs for my Desktop.

Backing up present working directory
hdir=$(pwd)

Updates/Upgrades
sudo apt-get --yes update

Utils
sudo add-apt-repository ppa:danielrichter2007/grub-customizer -y sudo apt-get --yes update sudo apt-get --yes install grub-customizer sudo apt-get --yes install tree sudo apt-get --yes install sshfs sudo apt-get --yes install nautilus-open-terminal sudo apt-get --yes install default-jre sudo apt-get --yes install default-jdk sudo apt-get --yes install virtualbox sudo apt-get --yes install git sudo apt-get --yes install nautilus-dropbox sudo apt-get --yes install inkscape sudo apt-get --yes install python-pip sudo apt-get --yes install python-sklearn sudo apt-get --yes install python3-pip sudo apt-get --yes install python-pyx sudo apt-get --yes install youtube-dl sudo pip install svgwrite

Git config
git config --global user.email "gilbertoamarcon@gmail.com" git config --global user.name "Gilberto Marcon"

Terminator
sudo apt-get --yes install terminator sudo cp -Rf terminator ~/.config/ sudo chmod +x ~/.config/terminator/

Sublime
wget -P ~/Downloads/ https://download.sublimetext.com/sublime-text_build-3126_amd64.deb sudo dpkg -i ~/Downloads/sublime-text_build-3126_amd64.deb rm ~/Downloads/sublime-text_build-3126_amd64.deb subl pkill sublime_text cp sublime-text-3/* ~/.config/sublime-text-3/Packages/User/ cp mimeapps.list ~/.local/share/applications/

Chrome
wget -P ~/Downloads/ https://dl.google.com/linux/direct/google-chrome-stable_current_amd64.deb sudo dpkg -i ~/Downloads/google-chrome-stable_current_amd64.deb rm ~/Downloads/google-chrome-stable_current_amd64.deb

Color Terminal
sed -i '/force_color_prompt=yes/s/^#//g' ~/.bashrc

Calc
mkdir -p ~/dev/ git clone https://github.com/gilbertoamarcon/calc.git ~/dev/calc/ cd ~/dev/calc/ ./build.sh sudo ./install.sh cd $hdir rm -Rf ~/dev/calc/

Cleaning up
sudo apt-get --yes autoremove

Removing Examples folder
rm -Rf ~/examples.desktop

Restarting the machine
sudo reboot
