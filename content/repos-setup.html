Repos Setup


This script will install and configure all repos.

i-CiFHaR


Cloning and building
rm -Rf ~/dev/icifhar > /dev/null 2>&1 git clone https://gilbertoamarcon@bitbucket.org/aldukeman/icifhar.git ~/dev/icifhar/ cd ~/dev/icifhar/ qmake make

Adding the lib folder to $LD_LIBRARY_PATH:
printf "\n# i-CiFHaR:\nexport LD_LIBRARY_PATH=~/dev/icifhar/lib\n" >> ~/.bashrc

MACPDDL

rm -Rf ~/dev/macpddl > /dev/null 2>&1 git clone https://gilbertoamarcon@bitbucket.org/aldukeman/macpddl.git ~/dev/macpddl/ cd ~/dev/macpddl/ ant

i-MAPC


Cloning and building
rm -Rf ~/dev/imapc > /dev/null 2>&1 git clone git@bitbucket.org:gilbertoamarcon/imapc.git ~/dev/imapc/ cd ~/dev/imapc/ touch variables.cmake cmake . make

Cloning and building
git clone https://gitlab.com/human-machine-teaming/planning/imapc ~/dev/imapc/
cd ~/dev/imapc/
git submodule init
git submodule update
cmake .
make
