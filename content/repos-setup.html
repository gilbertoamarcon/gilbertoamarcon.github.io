Repos Setup

This script will install and configure all repos.

i-CiFHaR

Cloning and building
rm -Rf ~/dev/icifhar > /dev/null 2>&1 git clone git@gitlab.com:human-machine-teaming/planning/icifhar.git ~/dev/icifhar/ cd ~/dev/icifhar/ qmake make
Adding the lib folder to $LD_LIBRARY_PATH:
printf "\n# i-CiFHaR:\nexport LD_LIBRARY_PATH=~/dev/icifhar/lib:$LD_LIBRARY_PATH\n" >> ${HOME}/.bashrc

MACPDDL

rm -Rf ~/dev/macpddl > /dev/null 2>&1 git clone git@gitlab.com:human-machine-teaming/planning/macpddl.git ~/dev/macpddl/ cd ~/dev/macpddl/ ant

i-MAPC

Cloning and building
rm -Rf ~/dev/imapc > /dev/null 2>&1 git clone --recursive git@gitlab.com:human-machine-teaming/planning/imapc.git ~/dev/imapc/ cd ~/dev/imapc/ cmake . make

Test Set

rm -Rf ~/dev/test-set > /dev/null 2>&1 git clone git@gitlab.com:human-machine-teaming/planning/test-set.git ~/dev/test-set/

git clone git@gitlab.com:human-machine-teaming/planning/test-results.git ~/dev/test-results/
