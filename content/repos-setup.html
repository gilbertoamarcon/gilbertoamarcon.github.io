Repos Setup


This script will install and configure all repos and all their dependencies.

General dependencies


Repos folder
sudo apt-get --yes install git mkdir -p ~/dev/

cmake install/upgrade (3.5.1):
sudo add-apt-repository ppa:george-edison55/cmake-3.x -y sudo apt-get --yes update sudo apt-get --yes install cmake sudo apt-get --yes upgrade cmake wget -P ~/Downloads/ http://www.cmake.org/files/v3.5/cmake-3.5.1.tar.gz tar -xvzf ~/Downloads/cmake-3.5.1.tar.gz -C ~/Downloads/ cd ~/Downloads/cmake-3.5.1 ./configure make sudo make install sudo update-alternatives --install /usr/bin/cmake cmake /usr/local/bin/cmake 1 --force cmake --version

Upgrade to gcc/g++ 5.4.0 (5.4.0):
sudo add-apt-repository ppa:ubuntu-toolchain-r/test -y sudo apt-get --yes update sudo apt-get --yes install gcc-5 g++-5 sudo update-alternatives --install /usr/bin/gcc gcc /usr/bin/gcc-5 60 --slave /usr/bin/g++ g++ /usr/bin/g++-5

Add a backup version of gcc/g++ sudo apt-get --yes install gcc-4.8 g++-4.8 sudo update-alternatives --install /usr/bin/gcc gcc /usr/bin/gcc-4.8 60 --slave /usr/bin/g++ g++ /usr/bin/g++-4.8
sudo update-alternatives --config gcc
Proto
sudo apt-get --yes install autoconf automake libtool curl make g++ unzip wget -P ~/Downloads/ https://github.com/google/protobuf/releases/download/v3.0.0/protobuf-cpp-3.0.0.tar.gz tar -xvzf ~/Downloads/protobuf-cpp-3.0.0.tar.gz -C ~/Downloads/ cd ~/Downloads/protobuf-3.0.0/ ./autogen.sh ./configure make make check sudo make install sudo ldconfig # refresh shared library cache.

Java 8

i-CiFHaR


Installing dependencies
sudo apt-get --yes install qt5-default

Clonning repository
git clone https://gilbertoamarcon@bitbucket.org/aldukeman/icifhar.git ~/dev/icifhar/

Building
cd ~/dev/icifhar/ qmake make

Adding the lib folder to $LD_LIBRARY_PATH:
printf "\n# i-CiFHaR:\nexport LD_LIBRARY_PATH=~/dev/icifhar/lib\n" >> ~/.bashrc
